# Retail Promotion Effectiveness Analysis

**Business Question:** Do promotional discounts drive long-term 
customer value at Olist — or do they attract one-time buyers 
and erode margins?

**Tools:** SQL · Python · Tableau  
**Dataset:** Olist Brazilian E-Commerce (Kaggle, 100k+ orders)  
**Status:** 🔄 In Progress

## Key Questions
- Do discount-acquired customers repeat purchase at lower rates?
- What is the 90-day LTV gap between discount vs full-price buyers?
- Which product categories should reduce promotional depth?

## Structure
- `/sql` — data prep and metric queries
- `/notebooks` — EDA, repeat purchase analysis, regression
- `/output` — dashboard screenshots, executive summary
